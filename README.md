# Esercizio Documentazione

Repository realizzata per gli esercizi del corso FISM 2026 dedicati a Git, GitHub, HTML e documentazione di progetto.

## Obiettivo

L'obiettivo del progetto è esercitarsi nella creazione e gestione di una repository GitHub, nella documentazione tecnica e nello sviluppo di semplici pagine web HTML e CSS.

## Funzionalità principali

* Gestione repository Git e GitHub
* Documentazione in formato Markdown
* Pagine HTML di esempio
* Utilizzo di CSS per la formattazione grafica
* Organizzazione dei file secondo una struttura di progetto
