# Gprocedura installazione
da locale a condiviso

## Installazione Repository da Termonale
0. aprire il termonale
1. clonare la repository `git clone https://github.com/jderia/TEST_FISM.git`
2. entrare nella cartella `cd TEST_FISM2026`

## Apertura della pagina Web da termonale

0. aprire il terminale
1. entrare nella cartella dove si è scarcato il progetto
2. `cd src`
3. `cat index.html`
